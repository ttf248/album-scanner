# 相册扫描器 (Album Scanner)

一个采用现代iPhone风格设计的图片相册管理工具，支持瀑布流展示、批量扫描、预览和管理本地图片文件夹。

## 🎨 设计特色

### iPhone风格界面
- 🍎 原生iOS 17设计语言
- 🌊 瀑布流相册展示
- 🎭 毛玻璃效果和圆角设计
- ✨ 流畅动画和悬停效果
- 📱 响应式布局适配

### 启动页设计
- 🚀 现代化启动界面
- 📋 快速操作卡片
- 🎯 直观的视觉引导
- 💡 智能功能推荐

## 功能特性

### 🔍 相册扫描
- 递归扫描指定文件夹中的所有图片文件
- 支持常见图片格式：JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP
- 自动识别包含图片的子文件夹作为相册
- 瀑布流展示相册缩略图、图片数量和文件夹大小
- 智能路径验证和错误处理

### 🖼️ 图片查看
- 高质量图片预览，支持缩放和全屏查看
- 键盘导航：左右箭头切换图片，ESC退出全屏
- 鼠标操作：滚轮缩放，拖拽平移
- 智能缩放算法，保持图片清晰度
- 支持GIF动图播放

### ⭐ 收藏管理
- 收藏喜欢的相册，一键访问
- 收藏状态持久化保存
- 收藏列表快速浏览
- iPhone风格收藏动画效果

### 📝 历史记录
- 自动记录最近浏览的相册
- 快速访问最近查看的内容
- 历史记录自动清理无效路径
- 智能路径验证

### 🎨 用户界面
- iPhone风格Material Design界面
- 瀑布流响应式布局
- 状态栏显示操作反馈和统计信息
- 直观的卡片式相册展示
- 优化的用户体验和交互反馈

### ⚙️ 配置管理
- 自动保存用户设置和偏好
- 记住上次浏览的路径
- 窗口大小和位置自动恢复
- 错误处理和配置备份

### ⌨️ 快捷键支持
- `Ctrl+O`: 浏览文件夹
- `Ctrl+S`: 扫描相册
- `Ctrl+R`: 显示最近浏览
- `Ctrl+F`: 显示收藏夹
- `F5`: 刷新扫描

## 使用方法

### 启动程序
```bash
python main.py
```

### iPhone风格启动页
1. **英雄区域**：应用图标和欢迎信息
2. **快速操作卡片**：
   - 📚 最近浏览：查看最近打开的相册
   - ⭐ 收藏夹：管理您收藏的相册
   - 🔍 智能扫描：自动发现图片文件夹
   - ⚙️ 设置：个性化您的体验
3. **路径输入区域**：选择和扫描相册文件夹

### 瀑布流相册浏览
- **自适应布局**：根据屏幕宽度自动调整列数
- **随机高度**：模拟真实瀑布流效果
- **卡片交互**：悬停效果和点击反馈
- **收藏功能**：一键收藏/取消收藏相册

### 基本操作
1. **选择相册文件夹**：点击"选择文件夹"按钮或使用快捷键 `Ctrl+O`
2. **扫描相册**：点击"开始扫描"按钮或使用快捷键 `Ctrl+S` 或 `F5`
3. **查看相册**：点击相册卡片上的"打开"按钮
4. **收藏相册**：点击相册卡片上的星形收藏按钮
5. **快速访问**：使用启动页的快速操作卡片

## 核心模块
