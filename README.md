# 相册扫描器 (Album Scanner)

一个基于Python和Tkinter的图片相册管理工具，支持批量扫描、预览和管理本地图片文件夹。

## 功能特性

### 🔍 相册扫描
- 递归扫描指定文件夹中的所有图片文件
- 支持常见图片格式：JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP
- 自动识别包含图片的子文件夹作为相册
- 显示相册缩略图、图片数量和文件夹大小

### 🖼️ 图片查看
- 高质量图片预览，支持缩放和全屏查看
- 键盘导航：左右箭头切换图片，ESC退出全屏
- 鼠标操作：滚轮缩放，拖拽平移
- 智能缩放算法，保持图片清晰度
- 支持GIF动图播放

### ⭐ 收藏管理
- 收藏喜欢的相册，一键访问
- 收藏状态持久化保存
- 收藏列表快速浏览

### 📝 历史记录
- 自动记录最近浏览的相册
- 快速访问最近查看的内容
- 历史记录自动清理无效路径

### 🎨 用户界面
- 现代化Material Design风格界面
- 响应式布局，适配不同屏幕尺寸
- 状态栏显示操作反馈和统计信息
- 直观的网格式相册展示

### ⚙️ 配置管理
- 自动保存用户设置和偏好
- 记住上次浏览的路径
- 窗口大小和位置自动恢复

## 使用方法

### 启动程序
```bash
python main.py
```

### 基本操作
1. **选择相册文件夹**：点击"浏览"按钮选择包含图片的根目录
2. **扫描相册**：点击"扫描"按钮开始扫描子文件夹
3. **查看相册**：双击相册卡片打开图片查看器
4. **收藏相册**：点击相册卡片上的收藏按钮
5. **快速访问**：使用"最近浏览"和"收藏夹"按钮

### 图片查看器操作
- **切换图片**：使用左右箭头键或点击导航按钮
- **缩放图片**：鼠标滚轮或点击缩放按钮
- **全屏查看**：双击图片或按F11
- **退出全屏**：按ESC键
- **拖拽平移**：在缩放状态下拖拽图片

## 核心模块

### ConfigManager (config.py)
- 用户配置的读取和保存
- 收藏夹和历史记录管理
- 设置持久化存储

### ImageProcessor (image_utils.py)
- 图片文件扫描和识别
- 图片尺寸和格式处理
- 文件夹大小计算
- 缩略图生成

### UI Components (ui_components.py)
- **StyleManager**: 界面样式管理
- **NavigationBar**: 顶部导航栏
- **AlbumGrid**: 相册网格展示
- **ImageViewer**: 图片查看器
- **StatusBar**: 状态栏
